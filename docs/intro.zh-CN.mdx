# 介绍

使用 iCraft Player 前端组件, 可以轻松地将 iCraft Editor 编辑的场景集成到您的项目中，并提供丰富的交互功能。

## 如何使用

首先，你需要安装 `@icraft/player` 包：

```bash
pnpm install @icraft/player --save
```

然后，在你的项目中导入 `ICraftPlayer` 组件，并传入 `fileUrl` 属性，值为 iCraft Editor 导出的
`.iplayer` 文件地址。

```jsx
import { ICraftPlayer } from "@icraft/player";

export default () => {
  return <ICraftPlayer fileUrl='*.iplayer' />;
};
```

在浏览器中打开你的项目，你应该可以看到 iCraft Editor 编辑的场景嵌入到页面中。
