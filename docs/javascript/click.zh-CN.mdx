# 点击事件

通过配置 `onClick` 参数，可以监听元素的点击事件，并执行相应的回调函数。

## 使用示例

下面是一个示例，展示了如何使用 iCraft Player 组件进行元素点击事件交互。


import { Wrap } from "../../demos";

<Wrap>
  <iframe src="/demos/click.html" style={{ width: "100%", height: "100%", border: "none" }} />
</Wrap>

```html copy
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta
      name="viewport"
      content="width=device-width, initial-scale=1.0"
    />
    <title>ICraft Player Click Demo</title>
    <style>
      body {
        margin: 0;
        padding: 0;
      }
      #container {
        width: 100%;
        height: 100vh;
        position: relative;
      }
      #text {
        position: absolute;
        bottom: 10px;
        right: 10px;
      }
    </style>
    <script src="https://unpkg.com/@icraft/player@latest/dist/umd/icraft-player.min.js"></script>
  </head>
  <body>
    <div id="container">
      <div id="text"></div>
    </div>
  </body>
  <script>
    const player = new ICraftPlayer({
      src: "/templates/AWSCloud.iplayer",
      container: document.getElementById("container"),
      onClick: (e) => {
        console.log(e);
        const { instance: element } = e;
        const name = element?.options?.name || "";
        document.getElementById("text").innerText = "Clicked:" + name;
      },
    });
  </script>
</html>

```

## API

|  Name   | description  |             type              |  default  | required |
| :-----: | :----------: | :---------------------------: | :-------: | :------: |
| onClick | 点击事件回调 | (params: ClickParams) => void | undefined |  false   |

### ClickParams
```ts
interface ClickParams {
  instance: Element3D | null;
  event: MouseEvent;
  data: OptionsType | null;
}
```
