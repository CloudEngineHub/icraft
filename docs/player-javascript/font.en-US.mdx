# Font

Since iCraft Editor's drawing font is based on WebGL, font resources need to be loaded separately. Especially in intranet environments, font resources need to be downloaded to your project in advance.

If font resources are not actively set, the default fonts from iCraft Editor server and Google will be used.

## Setting Font Resources

iCraft Editor has some built-in system fonts, with `AlibabaPuHuiTi` as the default font. You can set font resources through the `defineConfig` static method, choosing either `fontUrl` or `fonts`.

### fontUrl

You can set global font resources through `fontUrl`, which has higher priority than `fonts`. This is suitable for scenarios with only one font.

```ts
import { ICraftPlayer } from "@icraft/player";

ICraftPlayer.defineConfig({
  fontUrl: "https://unpkg.com/@icraft/engine/public/fonts/AlibabaPuHuiTi-3-85-Bold.ttf",
});
```

### fonts

You can set font resources through `fonts`, which has lower priority than `fontUrl`. This is suitable for scenarios with multiple fonts.

```ts
import { ICraftPlayer } from "@icraft/player";

ICraftPlayer.defineConfig({
  fonts: [
    {
      name: "AlibabaPuHuiTi",
      styles: [
        {
          weight: "Bold",
          fontUrl: "https://unpkg.com/@icraft/engine/public/fonts/AlibabaPuHuiTi-3-85-Bold.ttf",
        },
      ],
    },
  ],
});
```

<Callout type="warning">
  Note: defineConfig is a static method that needs to be called at the outermost level and at the beginning
</Callout>
