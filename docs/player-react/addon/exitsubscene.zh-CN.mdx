# 退出子场景控件

退出子场景控件，在子场景当中才会显示

## 如何使用

- 通过配置 `addons={["ExitSubScene"]}` 快捷启用子场景控件功能
- 或通过手动引用 `import { ExitSubScene } from "@icraft/player-react"` 手动启用子场景控件功能，手动启用可以更灵活的控制子场景控件的样式

<Callout type="info">
  一般情况下，进入子场景控件需要和退出子场景控件同时使用
</Callout>

import { Wrap, EnterSubSceneDemo } from "../../../demos";

<Wrap>
  <EnterSubSceneDemo />
</Wrap>

如上面的示例，点击 cache 元素时显示进入子场景控件，点击进入子场景控件后进入 cache 元素的子场景，进入子场景后会显示退出子场景控件，点击退出子场景控件时会回退到上一级场景中。

```tsx copy
import { ICraftPlayer } from "@icraft/player-react";

const style = {
  width: "100%",
  height: "100%",
  position: "relative" as const,
  overflow: "hidden" as const,
};

export default () => {
  return (
    <div style={style}>
      <ICraftPlayer
        src='/templates/AWSCloud.iplayer'
        addons={["EnterSubScene", "ExitSubScene"]}
      />
    </div>
  );
};
```

## API

|   Name    | description |        type         | default | required |
| :-------: | :---------: | :-----------------: | :-----: | :------: |
|   style   |    样式     | React.CSSProperties |         |  false   |
| className |    类名     |       string        |         |  false   |
