# Release Notes - 1.8.0 (2025-12-3)

![iCraft Editor](https://raw.githubusercontent.com/gantFDT/icraft/main/public/images/banner.jpg)

## Editor
### New Features
- Added support for JSON data plotting and JSON import/export
- Added editor Resource panel configuration functionality
- Added editor Properties panel configuration functionality
- Added editor Preset field configuration functionality

### Improvements
- Improved template loading speed

### Bug Fixes
- Fixed an issue where some models lost local structures during automatic model optimization
